Introduction

Thank you for downloading our theme.

This documentation presents information how to install and use this theme.

Welcome to Fashionist WordPress Theme!

Thank you for purchasing Fashionist Wordpress theme. Before you get started, please be sure to always check out this documentation. We outline all kinds of good information, and provide you with all the details you need know to use Fashionist Theme.

Compatible with

Wordpress 4.6 + Woocommerce 2.6 + WPBakery Visual Composer 4.11 +

Wordpress Information

To install this theme you must have a working version of Wordpress already installed. For information in regard to installing the WordPress platform, please see the WordPress Codex

Theme installation via WordPress

Step 1 Login to Wordpress Dasrdoard.
Step 2 Go to Appearance > Themes. Click Add new link and Upload Theme, choice a Fashionist-v[x].zip and click Install Now.
Step 3 Go to Appearance > Themes. Activate The Fashionist theme.
Previous Next

Theme installation via FTP

Step 1 Connect to your FTP server, and navigate to your wordpress folder.
Step 2 Unzip Fashionist-v[x].zip package, you will be Fashionist folder.
Step 3 Upload Fashionist folder into wp-content/themes/ folder.
Step 4 You login to Wordpress Dasrdoard. Go to Appearance > Themes and Activate The Fashionist theme.

Plugin installation

After activate Fashionist theme, you'll get the following message:

Theme Options

Fashionist has an advanced Theme Options panel that is loaded with options. We have the options organized into logical tabbed sections, and each option has a description of what it will do on the front end. There are too many options to go over in these docs but please take some time and navigate through each tab to see what we offer. Go to Fashionist Options in your WordPress admin to access the theme options panel. You can also backup your theme options by clicking on the Import/Export tab. That is a great way to move your custom options from site to site. See below for screenshots of our theme options panel. Please note: we are always adding new options so these screenshots may not reflect the most up to date options.

Theme Update Via WordPress

The easiest way to manually update your theme is to upload it via the WP dashboard.

Step 1 Log in to your WordPress Administration Panel (Dashboard).
Step 2 Go to Apprearance > Themes.
Step 3 Deactivate the Fashionist theme by simply activating a different theme.
Step 4 Delete the Fashionist theme. Do not worry, your content will not be lost!
Step 5 Click Install Themes > Upload > Browse and select the zip file (Fashionist-v[x].zip) to upload.
Step 6 Once it uploaded, you activate it again.

Theme Update Via FTP

Step 1 Login your FTP and go to wp-content/themes/ folder.
Step 2 Delete the Fashionist folder (we recommend you to backup this folder).
Step 3 Unzip Fashionist-v[x].zip to get the Fashionist theme folder, then you upload this folder to wp-content/themes/
